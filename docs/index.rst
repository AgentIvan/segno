Segno - Python QR Code and Micro QR Code encoder
================================================

Pure Python QR Code generator with no dependencies.

Segno (Italian for "sign" / "symbol") generates QR Codes and Micro QR Codes
according to ``ISO/IEC 18004:2015(E)``.
